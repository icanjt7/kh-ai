
# AI Heritage Design — Netlify 전시 템플릿

- `index.html` 하나로 동작하는 정적 전시 페이지입니다.
- `assets/` 폴더의 이미지는 앞/뒤 전환(hover)으로 보이며, 모달에서 상세 스토리를 제공합니다.
- Netlify에 배포 시, 이 폴더 전체를 업로드하거나 GitHub에 푸시하여 연결하세요.

## 배포 방법 (가장 간단한 수동 업로드)
1) Netlify 로그인 → "Add new site" → "Deploy manually"
2) 이 폴더(`saeng-a-exhibit`)를 ZIP으로 압축하여 업로드
3) 사이트가 생성되면 도메인 별칭을 설정하세요 (예: `saeng-a.netlify.app`).

## 수정 포인트
- `index.html` 상단 `<title>`과 Hero 텍스트
- `#credit` 섹션의 이메일 주소
- 색상 및 폰트는 `/css/style.css`에서 커스터마이즈
